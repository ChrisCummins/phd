graph2dot: Convert a ProgramGraph message to GraphViz dot.

Usage:

    graph2dot [--node_label=<attribute>] [--stdin_fmt={pb,pbtxt}] < program_graph.pbtxt)

Where --node_label is the attribute to use for producing node labels. Possible values:
  --node_label=           No node labels
  --node_label=text       Use the Node.text field as node labels.
  --node_label=int64:foo  Use Node.features.feature "foo", interpreted as an int64 list.
  --node_label=float:bar  Use Node.features.feature "bar", interpreted as a float list.
  --node_label=bytes:car  Use Node.features.feature "car", interpreted as a bytes list.
